How it works:
For window_size=3 and values [2.5, -1.0, 3.8, 1.9, -0.4]:
- Position 0: avg([2.5, -1.0, 3.8]) = 1.77
- Position 1: avg([-1.0, 3.8, 1.9]) = 1.57
- Position 2: avg([3.8, 1.9, -0.4]) = 1.77
And so on...

Rationale: 
Smoothing with a sliding window is essential for identifying hydrophobic and 
hydrophilic regions while filtering out local noise and outliers. This is the 
standard approach in bioinformatics for sequence analysis. Larger windows 
reveal broader trends (membrane-spanning regions), while smaller windows show 
local variations.

The function now correctly implements the Kyte-Doolittle smoothing methodology 
as described in the original 1982 paper.
