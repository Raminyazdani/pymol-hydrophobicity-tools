Ramin Yazdani | pymol-hydrophobicity-tools | main | WIP(extraction): Add OUTPUT_PREFIX environment variable support

Added configuration support for customizable output file naming through 
environment variables.

Changes:
- Import os and sys modules for environment variable access
- Added OUTPUT_PREFIX = os.environ.get('HYDROPHOBICITY_OUTPUT_PREFIX', 'hydrophobicity')
- Default value 'hydrophobicity' if environment variable not set
- This will be used for naming all generated plot files

Rationale: Using environment variables instead of hardcoded values (like student 
names or IDs) makes the tool flexible and professional. Users can customize 
output filenames without modifying code:

  $ export HYDROPHOBICITY_OUTPUT_PREFIX=protein_1ubq
  $ python calculate_hydrophobicity.py 1ubq.pdb
  # Generates: protein_1ubq_plot_window5.png

This approach follows 12-factor app principles for configuration and makes the 
tool suitable for batch processing multiple proteins with distinct output names.
